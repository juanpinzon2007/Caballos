JUEGO DE CARRERA DE CABALLOS CON CARTAS

Resumen Del Proyecto
Este proyecto es una aplicacion web desarrollada en Java con Spring Boot.
Automatiza el juego de carrera de caballos usando una baraja francesa.
Cada As representa un caballo y en cada turno solo se mueve el caballo del palo de la carta revelada.

Mejoras Importantes De Esta Version
- Interfaz visual profesional.
- Simulacion animada por turnos.
- Correccion del problema visual donde parecia que todos los caballos avanzaban al tiempo.
- Historial detallado de cartas y movimientos.
- Nueva carrera con un solo boton.
- Documentacion integrada dentro de la aplicacion.

Metodologia Aplicada
Analisis:
Se identificaron los componentes del problema: cartas, palos, caballos, posiciones, turnos y meta.

Alternativas:
Se considero una app de consola y una app web. Se eligio una app web por presentacion, claridad y facilidad de despliegue.

Diseno:
Se modelaron las clases Carta, Caballo, Palo, TurnoJuego y ResultadoJuego. Se uso una API interna para entregar el estado completo de la carrera.

Implementacion:
Se desarrollo la simulacion completa, una interfaz renovada, animacion por turnos y un historial detallado de cada movimiento.

Pruebas:
Se verifico el mazo de 52 cartas, la creacion de cuatro caballos y el ganador al alcanzar la meta.

Conclusiones:
La solucion evidencia modelo de datos, estructuras, operadores y restricciones mediante una aplicacion funcional y visualmente mas clara.
